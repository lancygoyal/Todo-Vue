# Todo-Vue

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Includes

- Vue.js
- Vuex
- ESLint

    .

	│   ├── store				# Vuex folder
    |   ├── main.js				# Main File
    └── package.json           	# Node dependencies
